# This is a ReactJS based Admin Dashboard that keeps tracks of blog posts, comments, subscribers and more.

### This is a ReactJS based web application

To get started and run the app:

- Clone the project.
- Run `npm install` to install the corresponding node packages
- Run `npm start` to run the app on [http://localhost:3000](http://localhost:3000)
